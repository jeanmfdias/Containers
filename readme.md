# Containers

Project to containers sample for Docker, to agile a development environment.

- Jenkins
- MySQL 5.7
- MySQL 8.0
- NodeJS
- PHP 8.2
- PostgreSQL 17
- Redis
- RabbitMQ

# To start the containers

MySQL 5.7
``./containers --mysql 57``

PostegreSQL
``./containers --postgres``

MySQL 8
``./containers --mysql 8``

Redis
``./containers --redis``

# To stop the containers

Redis by example
``./containers --redis stop``
