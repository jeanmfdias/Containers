# Containers

Project to containers sample for Docker, to agile a development environment.

- Jenkins
- MySQL 5.7
- MySQL 8.0
- NodeJS
- PHP 8.2
- PostgreSQL 17
- Redis
- RabbitMQ

# To start the containers

MySQL 5.7
``sh startup.sh --mysql 57``

PostegreSQL
``sh startup.sh --postgres``

MySQL 8
``sh startup.sh --mysql 8``
